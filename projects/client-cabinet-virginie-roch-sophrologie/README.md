## 🎨 Personnalisation

### Couleurs

Modifiez les couleurs dans `tailwind.config.ts`:

```typescript
colors: {
  primary: {
    500: '#3b82f6',  // Bleu principal
    600: '#2563eb',
    // ...
  },
  accent: {
    500: '#ef4444',  // Rouge accent
    // ...
  }
}
```
